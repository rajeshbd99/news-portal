1.What is the differences between var, let, and const variable?

VAR: Var declarations are globally scoped or function scoped. That means if a variable is declared with var outside a function then it is in the global scope(accessible anywhere). Otherwise, if they are inside a function we can only access them inside the function.,
LET: It's an enhanced version of var, it solves all the problems that come with the var keyword. Like, let is a block-scoped variable and we cannot re-declared but can be updated(re-assigned).
CONST: Const declarations share some similarities with let declarations, but const variables cannot be updated or re-declared. It maintains the constant values. Unlike var and let, if we are using const then it must be initialized at the same time when we declared the variable.

2. What is the differences between map(),forEach(),filter() and Find()?

  • forEach()=> .forEach(), is used to execute the same code on every element in an array but does not change the array and it returns undefined.
  • filter()=> .filter() checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria.
  • map()=> .map() executes the same code on every element in an array and returns a new array with the updated elements.
  • find()=> .find() When you want to select a single element from an array.
